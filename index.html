دليل تثبيت Visual Studio Code

أشهر محرر أكواد لتعلم البرمجة (HTML, CSS, JavaScript)

1

رابط التحميل المباشر

ادخل على الموقع الرسمي واضغط زر Download لتحميل النسخة المناسبة لجهازك:

🔗 الموقع الرسمي لـ VS Code
2

خطوات التثبيت (Windows)

  • افتح ملف VSCodeSetup.exe بعد التحميل.
  • اضغط Next ووافق على الشروط (I accept the agreement).
  • هام جداً: أثناء التثبيت، تأكد من تفعيل هذه الخيارات:
    • ✅ Add "Open with Code"
    • ✅ Add to PATH
    • ✅ Create Desktop Icon
  • اضغط Install ثم Finish.
3

إضافة Live Server (ضرورية للمبتدئين)

هذه الإضافة تسمح لك بتشغيل صفحة HTML مباشرة في المتصفح لرؤية التعديلات فوراً.

  1. افتح برنامج VS Code.
  2. اضغط على أيقونة Extensions (أو Ctrl+Shift+X).
  3. ابحث عن: Live Server
  4. اضغط Install.
4

تشغيل أول ملف HTML

1. من قائمة File اختر New File واحفظه باسم index.html

2. اكتب الكود التالي:

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

3. اضغط كليك يمين داخل الكود واختر Open with Live Server.

5

تثبيت Prettier وتفعيل Auto Save

لتحسين تنسيق الكود وحفظ التغييرات تلقائيًا:

  1. افتح نافذة Extensions (Ctrl+Shift+X).
  2. ابحث عن Prettier - Code formatter ثم اضغط Install.
  3. اذهب إلى Settings (Ctrl+,).
  4. فعّل Editor: Format On Save و Files: Auto Save (اختر "afterDelay" أو "onFocusChange").
